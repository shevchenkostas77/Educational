Nested classes (Вложенные классы).
Внутри одного класса можно создать внутренний класс.

public class Test {   // внешний класс, еще называют top level class (самый внешний класс);
}

Класс Test принадлежит пакету – package nested_class (находится внутри пакета).
По аналогии с переменными, возможно создание внутри класса других классов, они будут называться nested classes.

Есть несколько типов nested классов:
1. Static class.
public class Test {	// внешний класс;
	static class StaticClass {		// статический внутренний класс;
	}
}

2.  Inner class.
public class Test {	// внешний класс;
	class InneClass { 	// non-static внутренний класс;
	}
}

3. Local inner class.
Создавать класс возможно внутри какого-либо блока кода, к примеру, внутри метода или же конструктора.
void method() {		// метод;
	class LocalInnerClass {	// локальный внутренний класс;
	}
}
4. Anonymous class.

Nested classes используют тогда, когда нам какой-то класс нужно тесно привязать к другому, к внешнему,
без которого этот внутренний класс никому не нужен и нигде не используется (это как правило).
Грубо говоря nested класс становится элементом outer класса. Все типы (их 4 типа) nested классов отличаются
друг от друга, но, как правило, nested классы имеют доступ к переменным outer класса, даже к переменны которые
помечены access modifier private.
