
Comparable - это интерфейс, который должен имплементироваться классом,
объекты которого вы будете сравнивать по умолчанию (более частая логика сравнения);

Comparator - это интерфейс, который должен имплементироваться классом (X),
который может быть использован при редком сравнении объектов Вашего класса (более редкая логика сравнения);
